# V0.3.2 작업 계획 — 점 기반 현장 행동

작성일: 2026-07-19
상태: 구현 및 자동 검증 완료

## 1. 문제

V0.3.1까지의 지역 사건은 지역별 설명은 다르지만 실제 조작은 두 보상 버튼 중
하나를 고르는 구조입니다. 사용자가 상황을 직접 해결하지 않으므로 지역과 사건의
차이가 플레이로 전달되지 않습니다.

이번 버전은 그림을 추가하는 작업이 아니라, 점과 선만으로 직접 조작하는 세 가지
현장 행동을 구현하는 수직 슬라이스입니다.

## 2. 구현 범위

| 지역 | 행동 | 사용자 조작 | 실패·회복 |
| --- | --- | --- | --- |
| 낡은 부엌 | 실타래 풀기 | 빛나는 시작점부터 실의 연결 순서를 따라 매듭을 누름 | 잘못 누르면 한 단계 후퇴 |
| 식료품 창고 | 선반 철탑 오르기 | 아래에서 위로 번갈아 놓인 발판을 순서대로 누름 | 잘못 누르면 한 단계 미끄러짐 |
| 동네 편의점 | 무한 통로 진입 | 중앙 신호와 같은 색으로 맥동하는 문을 찾아 누름 | 잘못된 문은 실수로 기록 |

세 행동은 공통 점 렌더러를 사용하지만 목표, 공간 배치와 실패 피드백을 다르게
구성합니다. 식당과 공장은 이번 수직 슬라이스의 검증 뒤 별도 행동으로 확장합니다.

## 3. 게임 상태

- 각 지역의 행동은 `field_action` 데이터로 정의합니다.
- 완료한 행동 ID는 `completed_field_action_ids`에 기록합니다.
- 반복 행동 대기 시간은 `next_field_action_unix`에 기록합니다.
- 새 영구 필드가 추가되므로 저장 스키마를 2에서 3으로 올립니다.
- 스키마 2 저장은 완료 행동 없음, 대기 시간 0으로 마이그레이션합니다.

## 4. 보상

- 행동 완료 시 현재 운반량과 지역 생산 배수를 사용한 운반 보상을 지급합니다.
- 첫 완료에는 데이터에 정의된 추가 운반 보상을 한 번만 지급합니다.
- 실수 횟수는 결과에 표시하지만 이번 버전에서는 보상을 차감하지 않습니다.
- 행동 완료 기록과 보상은 UI가 아니라 `GameManager`에서 확정합니다.

## 5. UI와 표현

- 하단에 `현장 행동` 진입 버튼을 추가합니다.
- 행동 패널은 제목, 상황 설명, 점 행동 화면, 현재 진행 문구와 나가기 버튼으로
  구성합니다.
- 쥐와 목표는 점, 경로와 철탑은 선, 통로는 원으로만 표현합니다.
- 색만으로 판정하지 않도록 활성 목표에는 크기 변화와 외곽선을 함께 표시합니다.
- 마우스와 터치 입력을 같은 좌표 판정 경로로 처리합니다.

## 6. 제외 범위

- 식당·공장용 추가 행동
- 실시간 물리 실타래와 로프 시뮬레이션
- 전투, 부상과 영구 손실
- 보육실과 직업 시스템
- 실제 지역·군사 경계 콘텐츠

## 7. 완료 조건

- 세 행동이 서로 다른 지역 데이터에서 열립니다.
- 사용자가 점을 직접 눌러야 완료되며 단순 결과 버튼으로 우회할 수 없습니다.
- 잘못된 입력과 완료 입력이 자동 테스트로 구분됩니다.
- 첫 완료 보상은 한 번만 지급되고 완료 ID와 대기 시간이 저장됩니다.
- 스키마 2 저장이 스키마 3으로 안전하게 이관됩니다.
- 768×1024 세로 화면에서 행동 패널이 잘리지 않습니다.
- Godot 전체 테스트, Web export, Web 검증과 프로덕션 빌드가 통과합니다.

## 8. 구현 결과

- 세 지역에 `field_action` 데이터를 추가하고 같은 조작의 문구 교체가 아니라
  실타래·철탑·색 신호 판정을 각각 구현했습니다.
- 마우스와 터치가 `DotActionView.submit_point()`의 같은 판정 경로를 사용합니다.
- 틀린 입력, 올바른 단계 진행과 세 행동의 완료를 자동 테스트했습니다.
- 첫 완료는 부엌 기본 상태에서 치즈 125, 반복 완료는 치즈 75로 검증했습니다.
- 완료 ID와 20초 대기 시간을 스키마 3으로 저장하고 이전 저장 마이그레이션을
  검증했습니다.
- 768×1024 행동 패널 경계, Godot 전체 테스트, 메인 씬 실행, Web export,
  PWA·서비스 워커·gzip WASM 검사와 Vinext 프로덕션 빌드를 통과했습니다.
