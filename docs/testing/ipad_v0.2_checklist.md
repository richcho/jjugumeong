## 2. Safari 첫 실행

| 절차 | 통과 기준 | 결과 |
| --- | --- | --- |
| 새 Safari 탭에서 주소 열기 | 검은 빈 화면이나 로딩 정지 없이 게임 진입 | |
| 상단 버전 확인 | `쥐구멍 r4 0.2.7` 표시 | |
| 한글 확인 | 네모 글자나 누락 glyph 없음 | |
| 첫 쥐 왕복 대기 | 치즈 입고 숫자와 잔액 증가 | |
| 이동 구역 한 번 터치 | 3초 속도 부스트가 한 번만 실행 | |

## 6. 완료 판정

- 모든 항목이 통과하면 기준 문서와 로드맵의 V0.2 상태를 `완료`로 변경합니다.
- 하나라도 실패하면 V0.2 상태를 유지하고 changelog에 재현 절차를 기록합니다.
- 실제 기기에서 수행하지 않은 빈 항목은 통과로 간주하지 않습니다.
