# 변경 기록

이 문서는 날짜별 변경 기록의 인덱스입니다. 현재 확정 구조는
`docs/jjugumeong_master_plan.md`에서 관리하고, 변경 이유와 검증 결과는 날짜별
문서에 기록합니다.

## 기록 규칙

- 의미 있는 기능, 수정, 밸런스, 문서 결정은 기록합니다.
- “무엇을”뿐 아니라 “왜”, “어떻게 검증했는지”를 남깁니다.
- 저장 스키마, 밸런스, 지역 데이터와 Web 빌드 변경은 관련 문서 갱신 여부를
  함께 확인합니다.
- 실제로 되돌린 변경은 삭제하지 말고 상태를 `폐기`로 남깁니다.
