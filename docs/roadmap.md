# 개발 로드맵

로드맵은 기능 수보다 플레이어가 체감하는 완성 순서로 관리합니다. 현재 버전의
완료 조건을 통과하기 전에는 다음 대형 시스템을 앞당기지 않습니다.

## 1. 현재·다음·장기

| 구분 | 버전 | 목표 |
| --- | --- | --- |
| 완료 | V0.2 | 코드·자동 검증과 사용자 iPad 실행 확인 완료 |
| 현재 | V0.3 Alpha | 지역 재방문, 도감, 첫 발견 보상과 점 기반 현장 행동 |
| 이후 | V0.4~V0.5 | 쥐 수집, 직업과 지하 마을 |
| 장기 | V0.6 이후 | 연구, 지도, 세력, 국가와 우주 문명 |

## 2. 버전별 계획

### V0.1 Alpha — 핵심 루프

상태: 구현됨

- 자동 왕복과 치즈 입고
- 속도, 운반량, 쥐 수, 쥐구멍 강화
- 누적 치즈 기반 5개 지역과 생산 보너스
- 황금치즈와 클릭 부스트
- 튜토리얼, 통계와 토스트
- 3초 자동 저장, Web 3중 백업, 백업 복구, 최대 4시간 오프라인 보상
- 전체 지역과 강화를 빠르게 확인하는 Alpha 테스트 보상 25배
- 한글 Web 폰트와 iPad PWA 실행
- Safari 가변 화면과 안전 여백 대응

완료 게이트:

- headless 테스트 통과
- iPad Safari에서 한글, 터치, 화면 배치 정상
- 저장 후 재실행과 오프라인 보상 정상

### V0.2 — 보이는 성장

상태: 완료

- 다음 보상 카드, 진행 막대와 보상 미리보기 — 구현됨
- 지역 해금 발견 연출과 즉시 저장 — 구현됨
- 낡은 부엌, 식료품 창고, 동네 편의점 우선 배경 — 구현됨
- 식당과 치즈 공장 배경 확장 — 구현됨
- 속도·운반 강화에 따른 목도리·가방 변화 — 구현됨
- 쥐구멍 레벨에 따른 보강·선반·조명 변화 — 구현됨
- 인구 증가를 보여 주는 대기·생활 쥐 — 구현됨
- 모바일 UI 정보 우선순위 정리 — 구현 및 가로·세로 자동 검증됨
- 강화 외형 티어 단일 규칙과 다음 보상 상태 API — 구현됨
- r4 0.2.7 Web 릴리스 산출물과 iPad 실기기 체크리스트 — 준비됨
- V0.3.1에서 내부 표시처럼 보인 절차적 장비·굴 장식·굴 앞 장식쥐는 제거,
  티어 기반 전용 아트는 후속 작업으로 전환

완료 게이트:

- 최소 3개 지역을 글자 없이도 구분 가능
- 강화 2종 이상이 수치와 외형을 함께 변경
- 다음 목표와 보상을 한 화면에서 확인
- 가로·세로 대표 iPad 화면에서 잘림 없음
- 기존 V0.1 저장 호환

### V0.3 — 지역 플레이

상태: 구현 중

- 지역별 고유 자원 표현 — 배경·데이터 구현됨
- 고양이, 쥐덫, 인간 등 짧은 위험 사건 — 지역별 1종 구현됨
- 두 선택지 기반 사건과 결과 — 안전 보상·짧은 가속 1차 구현됨
- 해금 지역 선택과 재방문 — 구현됨
- 지역별 도감과 첫 발견 보상 — 해금 지역 자원·위험·사건 표시와 최초 1회
  운반 3회분 추가 보상 구현됨
- 부엌 실타래·창고 철탑·편의점 무한 통로 직접 행동 — 구현됨
- 선택 지역, 발견 기록과 현장 행동 기록 스키마 3 저장·마이그레이션 — 구현됨
- 식당과 공장 고유 현장 행동 — 구현됨
- 사건 선택이 숙련도·위험도·안전 경로와 생산량을 바꾸는 지속 결과 — 구현됨
- 숙련 Lv.3 지역 전문 선택과 잠금 이유 표시 — 구현됨

완료 게이트:

- 지역마다 배경뿐 아니라 플레이 선택이 다름
- 위험이 무작위 손실이 아니라 대응 가능한 사건
- 지역 선택 상태의 저장·마이그레이션 검증

### V0.4 — 쥐 수집과 직업

상태: 계획

- 쥐 종류, 희귀도와 고유 특성
- 운반쥐, 탐험쥐, 건설쥐 등 초기 직업
- 배치와 도감
- 첫 영웅과 짧은 영웅 이야기

완료 게이트:

- 쥐 선택이 생산 또는 사건 대응 방식을 바꿈
- 도감이 실제 발견 기록과 연결
- 희귀도만 높고 역할이 같은 쥐를 양산하지 않음

### V0.5 — 지하 마을

상태: 계획

- 식량 창고, 휴식 공간, 터널 입구
- 인구와 기본 소비
- 건설 시간과 오프라인 진행
- 마을 화면과 생활 애니메이션

완료 게이트:

- 쥐구멍이 기능과 화면 모두에서 마을로 변화
- 건물이 기존 자원·직업 루프에 연결
- 방치 보상과 소비가 플레이를 강제로 되돌리지 않음

### V0.6 — 시간과 위험

상태: 장기

- 낮과 밤
- 계절
- 고양이, 폭우, 질병과 재난
- 준비 시설과 대응 선택

### V0.7 — 연구와 자동화

상태: 장기

- 연구 분기
- 에너지와 기계
- 자동 운송
- 기술 선택에 따른 문명 특성

### V0.8 — 세계 지도와 세력

상태: 장기

- 하수도와 지하철 지도
- 탐험과 지역 연결
- NPC 세력, 무역과 외교
- 플레이어 문명사 기록 시작

### V0.9 — 출시 준비

상태: 장기

- 전체 UI와 접근성
- 사운드와 연출
- 밸런스와 성능
- 설정, 저장 안정성과 장시간 테스트
- 플랫폼별 패키징

### V1.0 — 첫 정식 문명

상태: 장기

낡은 부엌에서 지하 도시의 시작까지 하나의 완결된 성장 경험을 제공합니다.
V1.0 이후 국가, 세계 연합, 세대교체와 우주 시대를 별도 대형 단계로 확장합니다.

## 3. 장기 시대 계획

| 시대 | 핵심 경험 |
| --- | --- |
| 도시 시대 | 다중 지하 도시, 철도, 특화 건물 |
| 국가 시대 | 왕국, 국가 외교, 무역과 분쟁 |
| 세계 시대 | 대륙, 세계 경제와 전 세계 쥐 연합 |
| 진화 시대 | DNA, 돌연변이와 세대교체 |
| 우주 시대 | 달, 화성, 소행성 자원과 우주 기지 |
| 은하 시대 | 외계 세력, 은하 무역과 고유 문명사 |

장기 시대에 임의의 숫자 버전을 미리 고정하지 않습니다. 앞 단계의 플레이가
검증된 뒤 규모와 버전을 결정합니다.

## 4. 작업 우선순위 규칙

1. 실행 오류와 저장 손실 수정
2. iPad 화면과 조작 문제 수정
3. 현재 루프의 목표·보상 전달 개선
4. 시각 성장과 지역 차별화
5. 새 시스템 추가
6. 온라인과 대규모 확장
