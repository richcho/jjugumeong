# 2026-07-20 — 스코프 재정렬: 6시간 → 1일 목표, 시간예산 방법론 신설

## 배경과 결정

`docs/plans/scope_review_6h_target.md`(실행 전 상태)는 "6시간 안에 완전
정복 가능한 방치형 게임"을 목표로 스코프를 재정렬하자는 제안이었습니다.
조사 결과 이 "6시간" 수치는 실측된 적이 없고, master_plan/roadmap/
game_design/balancing 등 다른 어떤 정본 문서에도 등장하지 않는 미실행
제안값이었습니다.

사용자 결정으로 목표를 **"6시간" → "1일(연속 세션 기준 24시간)"**로
교체하고, 동시에 "전체 구현계획을 다시 짜자, 지금까지는 너무
주먹구구식이었다"는 요청에 따라 향후 모든 버전 계획에 적용할 시간예산
기반 공식 방법론을 신설했습니다. 장기 시대(도시·국가·세계·진화·우주·은하
6종)는 예산이 늘어난 것과 무관하게 스코프 밖 부록으로 유지합니다(확정
스코프는 V1.0까지).

작업 중 추가로 제기된 게임 경험 정합성 문제(시스템 간 연계 부족, 쥐
개체별 개성/독립성 부족, 의도된 재미 요소 정렬, 사용자 능동 참여 부족,
이동 목표지점 정확도)도 같은 재정렬 작업에 원칙으로 편입했습니다.

이 작업은 **문서 전용 재설계**입니다. 실제 24시간 실측 플레이나 코드
변경은 포함하지 않으며, 다음 단계로 명시적으로 미룹니다.
