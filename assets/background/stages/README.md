# V0.2 지역 배경

Godot 게임 화면에서 사용하는 1280×720 JPG 배경입니다. 생성 원본은 Web 배포
용량을 줄이기 위해 JPG로 축소했고, UI와 이동 쥐는 별도 코드 레이어로 그립니다.

| 파일 | 지역 | 핵심 방향 |
| --- | --- | --- |
| `old_kitchen.jpg` | 낡은 부엌 | 쥐 시점, 갈라진 벽, 따뜻한 부엌 |
| `food_storage.jpg` | 식료품 창고 | 선반, 마대, 푸른 저장 공간 |
| `convenience_store.jpg` | 동네 편의점 | 야간 냉장고, 청록·분홍빛 원정 |

생성 방식: OpenAI 내장 이미지 생성 도구

공통 프롬프트 방향:

```text
쥐의 지면 높이에서 본 16:9 스토리북풍 2D 게임 배경.
왼쪽은 쥐구멍, 오른쪽은 자원 지점, 중앙은 이동 공간으로 비운다.
상단 18%와 하단 24%는 UI가 겹치므로 핵심 사물을 피한다.
쥐, UI, 글자, 로고와 워터마크는 넣지 않는다.
```
